Data Visualization Projects

Below are two data visualization projects I worked on during my undergraduate studies. Three documents are involved in each project:

  • An HTML format of the data visualization report with interactive plots.
  • A CSV file of the data used in the visualization.
  • A PDF format of the report.

Catalan Referendum Tweet Sentiment Analysis Using Multiple Lexicons

This project involved analyzing Twitter data to understand public sentiment towards climate change over a six-month period. Using Python libraries such as Pandas, Matplotlib, and Seaborn, I cleaned the dataset, performed sentiment analysis, and created interactive visualizations to illustrate trends and patterns in the data.

  • HTML Report
  • Data CSV
  • PDF Report

Global Visa Cost Explorer: Visualizing International Visa Fees

This project uses the Global Visa Cost Dataset from the Migration Policy Center, European University Institute. It reports visa fees between country pairs for seven categories in 2019, collected mainly from government sites. Three interactive visualizations are included: a Sankey diagram, a bar chart, and a choropleth map.
